# Imposter Local

Local pass-the-phone imposter game for a group.

## Run it

### Easiest

Double-click `run-localhost.bat`

### Or from terminal

```powershell
node server.js
```

Then open:

[http://localhost:3000](http://localhost:3000)

## How it works

- Enter player names or keep the defaults.
- Start a round.
- Pass the device one player at a time.
- Everyone except the imposter sees the same secret phrase.
- Each player gives one clue.
- Vote on the imposter.
- After the vote, reveal the answer screen.
